# Diagram iconography

The icon set used by [`../architecture-diagram.html`](../architecture-diagram.html)
(and its exported `.mp4` / `.gif`). Two badge logos plus the line-icons drawn
for each stage of the architecture diagram.

All line-icons use `stroke="currentColor"` — set the surrounding `color` (or an
inline `fill`/`stroke`) to theme them. They're tiny, hand-authored, and
dependency-free.

## Badges

| File | What |
|---|---|
| `orbit-badge.svg` | The **orbit platform** mark — planet + orbit rings + satellites. Canonical copy also at [`orbit-station/web/public/icons/orbit-badge.svg`](../../../orbit-station/web/public/icons/orbit-badge.svg). |
| `self-badge.svg` | The **self** mark — glowing core + concentric memory rings + a bright “new memory” spark. Same visual family as the orbit badge. Canonical copy also at [`orbit-station/web/public/icons/self-badge.svg`](../../../orbit-station/web/public/icons/self-badge.svg). |
